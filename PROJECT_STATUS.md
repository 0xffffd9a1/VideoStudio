# VideoStudio – Статус проекта на 2026-06-29

## Общая информация
Проект представляет собой десктопное приложение для обработки видео.  
Интерфейс: Eel (Python-бэкенд + HTML/CSS/JS фронтенд).  
Ядро: FFmpeg, yt-dlp, кастомные модули.

## Реализованные функции
1. **Плеер с метками** – загрузка локального видео, кнопки «Начало»/«Конец» автоматически заполняют поля времени.
2. **Источники** – локальные файлы, YouTube-ссылки.
3. **Режимы** – video+audio, audio only, video only (без звука).
4. **Аудиофильтры** – нормализация громкости (loudnorm), усиление (+50%), шумоподавление (anlmdn).
5. **Точность обрезки** – быстрая (-c copy) и точная (перекодирование).
6. **Прогресс-бар** – отображает процент при скачивании и обработке.
7. **Пакетная обработка** – очередь задач с возможностью добавления, удаления, запуска всей очереди.
8. **Пресеты** – сохранение текущих настроек под именем, загрузка, удаление.
9. **Настройки** – папка сохранения, качество YouTube (best/1080p/720p/audio_only), тема (dark/light).
10. **Адаптивный интерфейс** – две панели (левая: плеер, вкладки «Файл», «Очередь», «Пресеты»; правая: параметры, лог).
11. **Логирование** – Python-лог в файл videostudio.log и в UI-консоль.
12. **Защита от дублирования сообщений** – при сохранении настроек проверяется реальное изменение.

## Известные проблемы / TODO
- При переключении темы в лог сыпались повторные сообщения – исправлено (проверка реального изменения + блокировка кнопки).
- Drag-and-drop полностью не работает (ограничение Eel). Можно реализовать через внешние библиотеки (pywin32).
- YouTube-видео не загружается в плеер (нужно скачивать и копировать во временную папку).
- Нет визуальной временной шкалы (Timeline) – следующий крупный шаг.
- Кнопки удаления задачи из очереди реализованы, но не всегда обновляются (зависит от вызова из Python).

## Как продолжить разработку
1. Склонировать репозиторий.
2. Установить зависимости (`pip install -r requirements.txt`).
3. Запустить `python main.py`.
4. Для добавления новых функций:
   - API добавлять в `web_backend.py` (декоратор @eel.expose).
   - JS-функции – в `script.js` (eel.expose для вызова из Python).
   - UI – в `index.html` и `style.css`.
5. При завершении сессии обновить этот файл.
