# 🎬 VideoStudio

Многофункциональная десктопная студия для обработки видео.  
Позволяет извлекать аудио/видео, нарезать фрагменты, применять аудиофильтры,  
работать с локальными файлами и YouTube. Интерфейс на базе Eel (Web + Python).

## Возможности
- 🎞️ Встроенный видеоплеер с установкой меток «Начало» / «Конец»
- ✂️ Обрезка видео (быстрая/точная)
- 🎧 Извлечение аудиодорожки
- 🎛️ Аудиофильтры: нормализация, усиление, шумоподавление
- 📦 Пакетная обработка (очередь задач)
- 🎚️ Сохранение/загрузка пресетов настроек
- 🌗 Тёмная/светлая тема
- 🖥️ Адаптивный интерфейс (две панели, вкладки)

## Установка и запуск
```bash
git clone https://github.com/твой-юзернейм/VideoStudio.git
cd VideoStudio
python -m venv venv
venv\Scripts\activate  # или source venv/bin/activate
pip install -r requirements.txt
python main.py
